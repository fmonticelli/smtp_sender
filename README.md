### SMTP Sender

Clone o repositório
```bash
git clone ht<span>tps://github.com/monticelli91/smtp_sender.git
```

Entre no diretório
```bash
cd smtp_sender/
```

Ajuste a permissão do script `smtp.sh` e execute
```bash
sudo chmod +x smtp.sh <br />
./smtp.sh <br />
```
